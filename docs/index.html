Encoding/Decoding Tool

This project is a Python application with a GUI for encoding and decoding text in various formats.

Features

  • Encode and decode text in various formats.
  • Auto-detect encoding format.

Installation

    1. Clone the repository:
    git clone https://github.com/yourusername/encoding-decoding-tool.git

    2. Navigate to the project directory:
    cd encoding-decoding-tool

    3. Install dependencies:
    pip install -r requirements.txt

    4. Run the application:
    python main.py
